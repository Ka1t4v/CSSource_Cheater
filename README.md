# CSSource_Cheater
基于GDI绘图实现的Counter-Strike:Source方框透视demo

游戏坐标，FOV等关键基地址通过CE7.4进行定位查找

实现算法根据捕获到的基地址通过GDI绘图，实现bot的方框透视

测试游戏版本为steam最新版Counter-Strike:Source（截止到2022/7/1）

只测试了单机bot版本，在线模式并未测试
